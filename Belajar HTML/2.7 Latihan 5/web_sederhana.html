Belajar Web Programming Dasar : HTML

Diposting oleh Fajar Maulana Sikumbang, 20 September 2023

Semua halaman web dan aplikasi web modern dibangun menggunakan 3 teknologi dasar: HTML, CSS, Javascript.

Ketiganya merupakan bahasa yang digunakan oleh web pada postingan kali ini, kita akan fokus pada HTML, dan kenapa kita harus mempelajarinya.

Apa Itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML merupakan sebuah bahasa markup yang mana digunakan oleh pengembang atau developer web untuk membangun struktur dan menampilkan konten dari sebuah halaman web. HTML bukanlah bahasa pemrograman karena didalamnya tidak memuat logika pemrograman.

HTML terdiri dari beberapa elemen yang menggambarkan berbagai macam konten, seperti paragraf, link, heading, gambar, video, dan lain-lain. Web browser mengenali HTML dan akan me-render kode HTML sebagai halaman web.

Di dalam HTML, masing-masing elemen terdiri dari 3 bagian:

  1. Tag Pembuka
  2. Tag Penutup
  3. Isi Konten

Kenapa kita harus belajar HTML?

Sebenernya banyak sekali alasan mengapa kita harus mempelajari bahasa dasar dari pemrograman web ini. Namun setidaknya ada 5 alasan kenapa kita harus mempelajarinya:

  • Supaya bisa menggunakan bahasa dasar untuk membangun web
  • Untuk membangun website yang bagus dan enak dilihat
  • Supaya bisa membangun aplikasi web (tertentu)
  • Supaya bisa pamer sama teman-teman
  • Supaya have fun
